# ROO
## Rust Object Oriented
 * Rust like syntax
 * Static typing
 * Made for OO
 * Borrow checker 

## Requirement
 * Linux
 * lcc (LLVM)
 * gcc

## Installation
Clone the repos</br>
build with `make` </Br>

## usage
```
$./roo file
```
file being the file to compile. </br>
Then run
```
$./run.sh
```
to compile the llvm file into an executable and launch it.
The output will be ./a.out
